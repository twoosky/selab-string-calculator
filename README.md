# SE-LAB Calculator

### 1. 요구사항
> 추가적인 기능 구현 환영! 물론, 필수적인 구현은 무조건~
1. 덧셈, 뺄셈, 나눗셈, 곱셈이 가능한 계산기를 구현합니다.
2. 입력은 `30 + 20 / 2 * 4` 순으로 받으며, 연산자 우선순위 없이 앞에서 부터 계산을 진행합니다.=
3. 출력은 `100`으로 나옵니다.
